# Android App Security Training – Dino Game

Небольшое Android-приложение на Java + Kotlin/Jetpack Compose, которое используется **не для продакшена**, а как учебный полигон для отработки навыков анализа защищенности Android-приложений.

## Цели проекта

- Тренировка базовых навыков реверса и анализа APK (static/dynamic analysis).
- Отработка типичных техник:
    - просмотр структуры APK и ресурсов;
    - поиск и анализ строк/констант;
    - базовый анализ сетевого трафика;
    - работа с `SharedPreferences` и локальными данными и т. п.
- Минимальная, намеренно простая кодовая база, чтобы фокус был на security-анализе, а не на сложной архитектуре.

## Технологии

- Язык: Java + Kotlin
- UI: Jetpack Compose
- Целевая платформа: Android

## Статус

Проект в активной разработке как учебный.  
API/функциональность могут часто меняться без сохранения обратной совместимости.
